ℹ️ How it works

1. Zero Cloud Lag: Facial recognition scans run directly in your browser using neural networks. No cameras or windows open on the remote server.

2. Consensus Guarantee: The scanner must consistently match a face for 5 consecutive frames at 85%+ accuracy before marking attendance to prevent any false-positives.

3. Auto-Saving: Once verified, attendance is pushed securely to the cloud database and logged immediately.
